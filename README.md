# Alfood
Projeto Front-end em andamento de um site de restaurantes, onde o usuário será capas de cadastrar um restaurante e seus pratos
## Tecnologias:
### Javascript;
### React.js;
### Sass.css;
### axios;
### Docker(para rodar a api)
